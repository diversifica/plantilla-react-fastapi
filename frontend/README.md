# Frontend

React 19 + Vite 7 listo para integrarse con FastAPI.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run test`
- `npm run validate`

## Arquitectura
- `/src/components`: UI compartida.
- `/src/pages`: vistas y rutas.
- `/src/services`: clientes API.
- `/src/tests`: Vitest + Testing Library.

## Variables
Configura `VITE_API_URL` para apuntar al backend (`http://localhost:8000`).
